# blog-mongo

Blog being built with the technologies below: 

- NodeJS; 
- ExpressJS; 
- Handlebars; 
- Bootstrap; 
- MongoDB; and
- Vanilla Javascript.


## Routes

These are the routes used so far.

- /admin/index
- /admin/posts
- /categorias
- /categorias/add
- /categorias/edit


More routes will be added soon. :D

## Dependencies

These are the dependencies used so far.

- connect-flash
- express
- express-handlebars
- express-session
- mongoose
